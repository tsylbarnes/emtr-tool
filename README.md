# EMTR Scenario Family Tool

Developed @ The Treasury - New Zealand

## Deployment
1. Use `shinylive::export(appdir = "app", destdir = "docs") to generate shinylive output`
2. Use `httpuv::runStaticServer("docs")` to run the static web app
